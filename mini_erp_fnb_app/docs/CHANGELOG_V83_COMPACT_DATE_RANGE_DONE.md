# V83 - Compact Date Range Calendar and Done-to-Apply Behavior

## Changes

- Made the table filter date range calendar smaller and less visually heavy.
- Removed the duplicated inner month title inside the calendar panel.
- Calendar date selections are now staged first.
- Clicking a date no longer applies the hidden filter value immediately.
- User must click **Done** to close the calendar and apply the selected date/range to the filter field.
- The table itself is still filtered only when the user clicks **Apply Filter** in the filter modal.

## SQL Migration

No SQL migration is required.
