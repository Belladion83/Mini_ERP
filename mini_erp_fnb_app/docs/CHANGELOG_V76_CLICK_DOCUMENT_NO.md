# V76 - Click Document Number to Open Transaction

## Changes

- PR/PO result tables no longer require radio selection + Open Selected Transaction.
- Users open PR/PO directly by clicking the document number.
- The same pattern should be used for all transaction/query/report tables going forward.
- Actions such as Edit, Release, Cancel, Create PO, Receive remain inside the transaction detail screen.

## No SQL migration

This version only changes UI/templates and purchase dashboard query output.
