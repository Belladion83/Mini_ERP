# V89 - View Mode Links & Compact Sale Channel Picker

## Changes

- Master Data codes and PR/PO document numbers now open records in View mode first.
- View mode locks all fields and shows an Edit button where editing is allowed.
- Edit mode keeps immutable fields locked, including master codes and document numbers.
- Clickable Master Codes / Document Numbers are bold links without arrow icons.
- Item Master > Sales > Opened Sale Channels now displays one representative selected Sale Channel in the field.
- Clicking Opened Sale Channels opens a checkbox picker showing all Sale Channels with selected items checked.

## SQL Migration

No SQL migration is required.
