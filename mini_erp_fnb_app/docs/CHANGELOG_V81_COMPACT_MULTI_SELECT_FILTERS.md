# V81 - Compact Searchable Multi-Select Table Filters

## Changes

- Replaced the always-visible checkbox list in table text filters with a compact searchable multi-select control.
- Users can search values in a field, select multiple values, remove selected chips, select all currently shown values, or clear the selection.
- Kept Date Range Calendar filters and number From/To filters from previous versions.
- Bumped static cache version to `v=81`.

## Migration

No SQL migration is required.
